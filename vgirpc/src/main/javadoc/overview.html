Transport-agnostic RPC framework built on Apache Arrow IPC serialization — the Java port of vgi-rpc, wire-compatible with the Python, Go, and C++ peers. Define RPC interfaces as ordinary Java interfaces: the framework derives Arrow schemas from your method signatures and record component types, and hands you a typed client proxy with automatic serialization. No .proto files, no codegen.

Serve an implementation with {@link farm.query.vgirpc.RpcServer} over stdio, AF_UNIX, or HTTP; connect with {@link farm.query.vgirpc.RpcConnection}. See the project README for a runnable example.
